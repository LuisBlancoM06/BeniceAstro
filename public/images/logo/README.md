# Logo de Venice

Esta carpeta contiene los archivos del logo de la tienda.

## Archivos sugeridos:

| Archivo | Uso | Formato |
|---------|-----|---------|
| `logo.svg` | Logo principal vectorial | SVG |
| `logo.png` | Logo en PNG con transparencia | PNG |
| `logo-white.svg` | Logo blanco para fondos oscuros | SVG |
| `favicon.ico` | Icono del navegador | ICO |

## Notas:
- Actualmente el logo es un SVG inline en el código
- Puedes añadir aquí tu logo personalizado y actualizar Layout.astro

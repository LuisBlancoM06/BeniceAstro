# Imágenes de Productos

Esta carpeta es para almacenar las imágenes de los productos de la tienda.

## Estructura sugerida:

```
productos/
├── perros/
│   ├── alimentacion/
│   ├── salud/
│   ├── accesorios/
│   └── juguetes/
├── gatos/
│   ├── alimentacion/
│   ├── salud/
│   ├── accesorios/
│   └── juguetes/
└── otros/
```

## Notas:
- Tamaño recomendado: 600x600px (cuadrado)
- Formato: JPG, PNG o WebP
- Fondo blanco o transparente preferido
- Nombra los archivos de forma descriptiva (ej: pienso-royal-canin-adulto.jpg)

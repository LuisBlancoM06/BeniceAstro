# Imágenes del Carrusel Hero (Página Principal)

Esta carpeta contiene las imágenes para el carrusel/slider de la página principal.

## Archivos necesarios:

| Archivo | Descripción | Tamaño recomendado |
|---------|-------------|-------------------|
| `perros.jpg` | Imagen de perros para el primer slide | 1200x400px |
| `gatos.jpg` | Imagen de gatos para el segundo slide | 1200x400px |
| `favoritos.jpg` | Imagen para el slide de favoritos | 1200x400px |

## Notas:
- Si no existen estas imágenes, se cargarán automáticamente desde Unsplash
- Formato recomendado: JPG o WebP
- Las imágenes deben ser horizontales (landscape)
- Optimiza el peso de las imágenes para mejor rendimiento
